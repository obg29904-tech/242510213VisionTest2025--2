# C++ 学习报告

## 1. 类模板

模板是创建泛型编程的基础。
- template <class type> class class-name {...}
type 为占位符类型名称，可以在类被实例化的时候进行指定。

## 2. virtual function

虚函数允许用基类的指针来调用子类的这个函数。只能借助指针或者引用来达到多态的效果。实现方法：
在定义前加上 virtual 。

## 3. std::weak_ptr

智能指针。定义位于 memory 头文件中。

## 4. friend function

类的友元函数是定义在类外部，但有权访问类的所有私有成员和保护成员。尽管友元函数的原型在类的定义中出现过，但并不是成员函数。实现方法：
在类定义中该函数前使用关键字 friend 。

## 5. static

- 局部静态变量在函数调用结束后不会被销毁，而是保持其值直到程序结束。
- 类的静态成员在类的所有实例之间共享，且在类加载时就存在与内存中。

## 6. std::thread

- std::thread : 用于创建和管理线程。
- std::mutex : 用于线程之间的互斥，防止多个线程同时访问共享资源。
- std::lock_guard & std::unique_lock : 用于线程间的条件变量，协调线程间的等待和通知。
- std::future & std::promise : 用于实现线程间的值传递和任务同步。
